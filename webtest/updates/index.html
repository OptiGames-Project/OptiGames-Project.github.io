Extended Kernel for Windows Vista: Prerequisite Updates

Click here to download Cumulative Update KB4474419.
Additional download options via the Microsoft Update Catalog are available here.


Click here to download Platform Update KB971512.
Additional download options via the Microsoft Update Catalog are available here.


Issues are likely to occur if the aforementioned updates are not installed:

KB4474419 must be installed prior to installation of the Extended Kernel.
The Cumulative Update is a prerequisite for the Extended Kernel. Issues are likely to occur at other update levels.
KB971512 is highly recommended, but not required.
Some applications, such as Firefox may encounter issues if the latter update is not applied.
